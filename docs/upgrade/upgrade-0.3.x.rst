Upgrade to 0.3.x
================

There are two upgrade paths to the SecureDrop 0.3.x release series:
one from SecureDrop 0.2.1, and one from SecureDrop 0.3pre (an
unofficial release that some instances used from November 2014 - March
2015).

Choose the appropriate guide below, based on the version of SecureDrop
you are currently running:

.. toctree::

   0.2.1-to-0.3.x
   0.3pre-to-0.3.x
